Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

AllMyGames

Bij dit project is het doel om een Angular frontend applicatie te realiseren in combinatie met een NoSQL database. Er horen 6 entiteiten bij dit project, namelijk: User, Gamelist, Game, Review, Developer en Adres. Het doel van mijn applicatie is dat gebruikers kunnen in verschillende lijsten (of 1 grote lijst) kunnen bijhouden welke games zij op het moment bezitten. Ook kunnen gebruikers reviews schrijven over deze games.

Entiteiten

Hier geef je een toelichting op je entiteiten. Voeg ook een afbeelding van je model toe zoals dat bij NoSql gevraagd wordt.

User

  • id: number
  • name: string
  • age: number
  • emailAdres: string
  • phoneNumber: string

GameList

  • id: number
  • name: string
  • description: string

Review

  • id: number
  • title: string
  • description: string
  • rating: number

Game

  • id: number
  • title: string
  • summary: string
  • rating: number
  • ageRating: string
  • price: number
  • completed: boolean

Developer

  • id: number
  • name: string
  • dateFounded: date

Adres

  • id: number
  • postalCode: string
  • street: string
  • houseNumber: number
